About

あさか

新潟を拠点に、中小企業のAI活用・業務システム刷新を支援しています。

「IT担当がいない」「昔作ったシステムが誰も触れない」「DXって何から始めればいい？」——そういう現場の悩みを、実装まで責任を持って一緒に解決します。

社内向けAIチャットボットの構築から、レガシーシステムのモダン化、業種特化のWebシステム開発まで対応します。

できること

  • AI活用導入（ローカルLLM・ChatGPT等）
  • 業務自動化・ワークフロー設計
  • レガシーシステムのモダン化
  • Webシステム開発（業種特化）
  • IT全般の相談・社内勉強会

サービス

AI導入支援

社内チャットボット、業務自動化、ChatGPT活用——目的に合わせたAIを選んで実装します。データが社外に出ないローカルAIにも対応。

レガシーシステム刷新

VB.NET、Access、Excelマクロ——「誰も触れない」システムをモダンな環境に移行。保守・拡張できる状態にします。

業務システム開発

紙・Excelでの業務をWebシステム化。業種・業務フローに合わせてゼロから設計・開発します。

IT相談・社内勉強会

「何から始めればいいか分からない」という段階からOK。現状整理・ロードマップ作成・社内勉強会の開催まで対応します。

実績

AI活用

社内向けAIチャットボット構築

ローカル環境（n8n + ローカルLLM）でAIチャットボットを構築。社内データが外部クラウドに送られないため情報漏えいリスクゼロ、月額費用なし。FAQ対応・業務マニュアル検索に活用可能。

システム刷新

VB.NET → C# 移行

将来的なサポート縮小リスクを抱えたVB.NETシステムをC#へ移行。既存の業務ロジックを維持しながら、保守性・拡張性を大幅に向上。

システム刷新

Microsoft Access → C# 移行

「誰かが昔作った、誰も触れない」Access業務システムをC#アプリに移行。複数人利用・クラウド連携・AI組み込みが可能な状態に。「Accessが壊れたら終わり」のリスクを解消。

システム開発

タクシー配車システム構築

中規模タクシー会社向けWebベース配車管理システムをゼロから開発。紙での配車管理をデジタル化し、手作業・記録ミスのリスクを解消。ブラウザから操作でき、台数拡張にも柔軟に対応。

AI活用

AIエージェント業務管理システム構築

Claude Code + MCP連携で構築した自律型の業務管理エージェントシステム。PM・営業・経理・クリエイティブなど8名の「AI社員」を設計・実装し、Google Calendar・Gmail・GitHubと連携したデイリーブリーフィング、週次評価、タスク管理を自動化。カレンダー・メール・タスク管理ツールとのAPI連携を含む実装例として、中小企業への導入パターンを具体的に示せる。
